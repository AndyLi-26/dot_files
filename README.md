# dot_files
nvim, tmux and initrc file

install neovim using on linux 
```
sudo apt get neovim
```
or
```
brew install neovim
```
Install Packer from  https://github.com/wbthomason/packer.nvim

copy .zshrc with care, only alias are useful mainly
```
mv dot_files/.zshrc ~/
mkdir .config
cp -r dot_files/nvim ~/config
```
